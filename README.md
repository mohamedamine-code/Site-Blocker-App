# Site Blocker App

Site Blocker is a Flutter-powered Android VPN application that lets you maintain a local blocklist of distracting or harmful websites. URLs are stored in a shared SQLite database, a cryptographically strong 16-character removal code is generated automatically for every entry, and an Android VPN service enforces the blocklist system-wide.

## Features

- 🛡️ **VPN-backed enforcement** – Android `VpnService` intercepts DNS/HTTP packets and stops traffic headed to blocked domains.
- 🗂️ **Local, tamper-resistant blocklist** – Sites live in a SQLite database; removal codes are hashed with SHA-256.
- 🔐 **Secure removal workflow** – Users see each autogenerated removal code once and must re-enter it later to unblock.
- 📱 **Clean Flutter UI** – Add, list, and remove blocked sites via dedicated screens with clear CTAs.
- 🚨 **Block alerts** – When traffic is blocked, the app surfaces a custom Flutter block screen (when in foreground) and posts a system notification.

## Requirements

- Flutter 3.19+ with the stable channel configured in your PATH.
- Android Studio or the Android SDK/command-line tools.
- A physical Android device or emulator running Android 8.0 (API 26) or newer.

## Getting Started

1. **Install dependencies**

   ```bash
   flutter pub get
   ```

2. **Run on a device**

   ```bash
   flutter run
   ```

   The first time you press “Start VPN” (or open the home screen), Android displays a system VPN permission dialog. Accept it to allow the service to run. If you deny it, return to the app and try again after granting permission.

## Using the App

1. Open the **Add Site** screen, enter a domain (e.g., `example.com`), and tap **Block Site**.
2. Copy the **one-time removal code** that appears. It is never stored in plain text, so losing it means the site stays blocked until you reinstall the app or clear data.
3. The VPN service begins intercepting requests and compares hosts against the blocklist. Matching traffic is dropped, the Flutter block screen is shown if the app is open, and a notification is posted system-wide.
4. To unblock, open **Remove Site**, paste the saved code, and confirm. The domain is deleted only when the hashed code matches what is stored.

## Security Notes

- Removal codes are generated with `Random.secure()` and hashed via SHA-256 before storage.
- Only generated codes are accepted—user-provided codes are rejected to avoid weak secrets.
- The Android VPN service and the Flutter app share the exact SQLite database file (`site_blocker.db`) so enforcement always reflects the latest data.
